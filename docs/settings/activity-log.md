# Activity Log

The **Activity Log** shows actions across your organization. It helps you understand who did what, when it happened, and which system events were recorded.

## What appears in the log

Activity entries can include:

- User name or system actor
- Action label
- Action type
- Timestamp
- Additional details when available

## Filtering

Use the action-type filter at the top of the page to narrow the log to a specific kind of activity.

## Loading more

The log loads recent activity first. Click **Load more** to continue reviewing older entries.

## Tips

- Use the Activity Log when investigating who changed or completed something.
- For transaction-specific context, open the related transaction and review its tabs alongside the activity trail.
