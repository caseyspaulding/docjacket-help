# Portal Links

The **Portal Links** page shows portal access across every transaction. Use it to monitor active links, extend expiring links, and revoke access when needed.

## What you see

Portal Links includes summary cards for:

- Total links
- Active links
- Total access count
- Buyer/seller and agent link counts
- Links expiring soon

The table shows each portal link's transaction, recipient, portal type, status, access count, last accessed date, and expiration date.

## Managing links

From the Portal Links page you can:

- Include or hide revoked links
- Filter by status
- Filter by portal type
- Extend expiring links
- Revoke links in bulk
- Open the related transaction

## When to use this page

Use the transaction's Portal tab when you are managing one deal. Use the Portal Links page when you need organization-wide visibility across all shared portal access.
