# E-Signatures

The **E-Signatures** page gives you one place to review signature requests and signing progress.

Signature requests are usually created from a transaction document. The E-Signatures page then tracks the request across statuses such as pending, partially completed, completed, cancelled, declined, or expired.

## What you see

The list includes:

- Document name
- Related transaction address
- Signature status
- Number of signers
- Completion progress
- Created date

## Cancelling a request

Pending or partially completed requests can be cancelled from the E-Signatures page. Completed requests remain as signed records.
