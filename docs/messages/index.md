# Email

DocJacket keeps communication attached to the transaction where the work is happening. Open a transaction's **Email** tab to review the transaction inbox, communication timeline, scheduled email, and internal notes for that deal.

![Transaction Email tab with Inbox, Timeline, Scheduled, and Notes views](/img/messages/email-inbox-list.png)

The transaction Email workspace includes:

- **Inbox** - Send, reply, forward, search, archive, draft, and review transaction email
- **Timeline** - See a chronological communication history and copy or create the deal email address
- **Scheduled** - Manage queued emails from task automation, key-date automation, and compose-time send later
- **Notes** - Keep rich internal notes, replies, and pinned updates on the deal

SMS and portal chat live in the transaction communication tools as a conversation view when enabled, but the day-to-day email workflow starts from the transaction's **Email** tab.

The **Inbox** item in the sidebar takes you to inbound work, not a global transaction email inbox. Transaction email history is managed inside each transaction workspace.
