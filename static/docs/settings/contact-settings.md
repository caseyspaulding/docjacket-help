# Contact Settings

Use **Settings > Contacts** to control how contacts are categorized and filtered across DocJacket.

## Contact types

Contact types define the labels available when adding people and companies to your contact list.

From the **Contact Types** tab, you can:

- Add a type with an internal name and display name
- Choose a color
- Mark a type as a service provider
- Hide inactive types from dropdowns
- Edit existing types
- Delete unused types

Types with existing contacts cannot be deleted until those contacts are moved or removed.

## Service providers

Turn on **Service Provider** for companies such as title, escrow, lenders, inspectors, and vendors.

Service provider types appear under organization and provider filters instead of being treated like individual people.

## Tags

Use the **Tags** tab to organize contacts and transactions with reusable labels.

You can:

- Create tags with a name, type, and color
- Search and filter tags
- Edit tag names and colors
- Delete tags that are no longer needed

Keep tags short and operational. They work best for repeatable filters such as VIP, Referral Partner, Builder, or Compliance Review.
