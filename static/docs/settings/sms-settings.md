# SMS Settings

Use **Settings > SMS** to manage your organization's texting setup, usage, and SMS credits.

## Business number

Paid organizations can use a dedicated business phone number for outbound SMS. If your plan does not include a dedicated number, DocJacket shows an upgrade prompt.

From SMS settings, you can:

- Search available phone numbers by area code
- Choose a number
- Set up the selected number
- See the city or region associated with the number when available
- Confirm which number is currently active

If you do not have a dedicated number, DocJacket can fall back to a shared DocJacket sending number.

## Sending and replies

To send a direct SMS from a transaction, open the transaction, go to **Messages > SMS**, and click **New Message**.

From the SMS tab you can send to one contact, send to multiple contacts, reply in an existing thread, attach a file, or use a saved SMS template.

Replies return to the same message thread automatically, so the conversation stays attached to the transaction.

Portal chat is separate from SMS and does not count against SMS usage. Contacts with portal access can reply through the portal instead of using text credits.

## Usage and credits

SMS settings show monthly usage and the number of texts available for the current billing period.

When you use the texts included in your plan, DocJacket shows an overage prompt. Click **Buy 250 More — $5** to purchase extra credits for the current month.

If your organization reaches the monthly hard limit, DocJacket blocks additional SMS sends and prompts you to contact support.

## Releasing a number

The **Danger Zone** appears when your organization has a dedicated number.

Releasing a number permanently gives it up. Contacts who saved that number will no longer recognize your texts, and you cannot get the same number back. Your account falls back to the shared DocJacket number after release.
