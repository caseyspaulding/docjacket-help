# Portal Links

The **Portal Links** page shows portal access across every transaction. Use it to monitor active links, extend expiring links, include revoked links, and revoke access when needed.

## What you see

Portal Links includes summary cards for:

- Total links
- Active links
- Total access count
- Buyer/seller and agent link counts
- Links expiring soon

The page also highlights links expiring within 7 days and recently accessed links.

The table shows each portal link's transaction, recipient, portal type, status, access count, last accessed date, and expiration date.

## Managing links

From the Portal Links page you can:

- Include or hide revoked links
- Filter by status
- Filter by portal type
- Extend expiring links
- Quick-extend a single active link by 7 days
- Select active links and bulk extend them
- Revoke links in bulk
- Open the related transaction
- Review recently accessed links and links expiring within 7 days

## When to use this page

Use the transaction's **Portal** tab when you are creating or previewing links for one deal. Use the transaction's **Contacts** tab when you are managing a specific transaction contact's portal link. Use the organization-wide **Portal Links** page when you need cross-transaction visibility, expiring-link review, or bulk portal-link cleanup.
