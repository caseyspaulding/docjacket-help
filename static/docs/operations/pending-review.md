# Review Queue

The **Review queue** is the topbar drawer for work that needs a quick decision.

Open it from the document-tasks icon in the topbar.

## What appears here

The drawer can include:

- **Pending Uploads** - extraction groups that finished scanning but have not been reviewed and saved
- **Ready for Review** - automation-rule drafts, email drafts, SMS drafts, and other actions waiting for approval or skipping
- **Scheduled** - upcoming reminders, scheduled emails, and scheduled messages
- **Done** - recent sent, approved, skipped, rejected, or completed activity

## Pending uploads

If the drawer shows **Pending Uploads**, those uploads were scanned but never saved into a transaction or listing.

Use **Review** to return to the extraction review flow. Use **Discard** only when the upload should stop appearing as unfinished work.

## Ready for Review

Use the **Ready for Review** tab for drafts and actions that need a human decision.

Depending on the draft, you may be able to:

- Open the detail view
- Edit supported email or SMS content
- Approve or send the draft
- Skip the draft
- Use **Skip past-deadline** for drafts whose deadline has already passed
- Use **Approve All** when every ready item should proceed

## Scheduled

Use the **Scheduled** tab to review upcoming sends.

Scheduled emails can be edited, sent now, or cancelled. Scheduled messages can be cancelled.

## Done

Use the **Done** tab to check recent activity from the last several days.

Done items can show what was sent, approved, skipped, rejected, or completed, along with the related transaction when available.

## If you have an old Pending Review link

Older **Pending Review** links now redirect to the Inbox. Use the topbar Review queue for current review work.
