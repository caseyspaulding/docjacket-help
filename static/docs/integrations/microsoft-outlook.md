# Microsoft Outlook

Connect Microsoft services from **Settings > Integrations**. Each Microsoft card has its own connection status and connect, reconnect, or disconnect action.

## What's included

### Outlook Mail

Connect Outlook Mail to send transaction email through Microsoft 365. Outlook Mail powers:

- Transaction email compose, reply, reply-all, and forward
- Scheduled one-off emails from the Email tab
- Scheduled emails from tasks, key dates, reports, and automation rules
- Sent-message history stored with the transaction in DocJacket
- Attachment sending from transaction documents or uploaded files

### Outlook Calendar

The Outlook Calendar card connects Microsoft calendar access from the integrations page. Use it when your workflow depends on Microsoft calendar permissions.

### OneDrive

The OneDrive card connects Microsoft file access from the integrations page. The current transaction document import and linked-folder workflow is Google Drive-specific.

## Connecting

1. Go to **Settings > Integrations**
2. Find the Microsoft service you want to connect
3. Click **Connect**
4. Sign in with your Microsoft account
5. Grant the requested permissions

If the card shows **Expired**, click **Reconnect** so DocJacket can refresh the permission.

## Tips

- Connect Outlook Mail if your organization sends transaction email from Microsoft 365.
- If Outlook Mail disconnects, reconnect it before relying on scheduled email sends.
- Gmail supports multi-account management in DocJacket today. Outlook account management is handled by reconnecting or disconnecting the Outlook Mail card.
