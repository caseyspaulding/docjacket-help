# Follow Up Boss

Use **Settings > Follow Up Boss** to connect Follow Up Boss and manage how data syncs into DocJacket.

## Tabs

The Follow Up Boss settings page includes:

- **Connections** — Add, review, and manage Follow Up Boss connections
- **Stages** — Review stage configuration
- **Mappings** — Map Follow Up Boss fields and stages to DocJacket fields
- **Automations** — Manage automation behavior tied to the integration
- **Activity** — Review recent sync events and connection activity

## Add a connection

1. Go to **Settings > Follow Up Boss**
2. Open the **Connections** tab
3. Click **New Connection**
4. Complete the setup modal
5. Save the connection

After connecting, review sync settings so contacts, deals, and stages move the way your team expects.

## Sync settings

Use sync settings to decide which Follow Up Boss records should create or update DocJacket data.

Review mappings before turning on broad sync. Clean mappings reduce duplicate contacts and make transaction automation more predictable.

## Activity

Use the **Activity** tab to confirm recent sync behavior and diagnose integration issues. If records do not appear as expected, check activity first, then confirm connection status and field mappings.
